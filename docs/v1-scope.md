# V1 Product Scope — TaskFlow

**Status:** Defining (in progress)

## Concept
WhatsApp AI bot as personal ops assistant. You chat → it organizes. It reminds → you act.

## Modules
1. Task Management (via chat + web dashboard)
2. Project Organization (via chat + web dashboard)  
3. Docs/Notes (via chat + web dashboard)

## Key Behaviors
- Smart message parsing (natural language → structured task)
- Proactive: morning briefs, deadline reminders, overdue alerts, weekly reports
- All notification timing user-customizable

💡 Key Design Decisions
📱
Phone number = user identity
No email/password required. WhatsApp number is the sole identifier.
💰
All money in integer paise
No floating point math. Every amount stored as paise for precision.
🧠
Hybrid parser: 80% regex + 20% LLM
Regex handles common patterns (free). LLM fallback costs ~₹0.008/call.
💬
WhatsApp replies < 300 chars
Short, scannable responses optimized for mobile chat UX.
⚡
Zero setup — first message = first task
No onboarding flow. Send a message, get started instantly.
🇮🇳
India-first: Hinglish, INR, WhatsApp-native
Built for how young Indians actually communicate and manage money.
